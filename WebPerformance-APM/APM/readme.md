* 用户希望了解问题的起因以及何时能解决他们的困扰。而程序员在面对用户最棘手问题的时候，需要做到快速定位问题根源，并给出明确的问题修复时间。
* 前端同学永远都是站在用户的最前面的人，而用户是产品最重要的资产之一。如何在有限的时间内通过前端全链路准确定位用户问题，并有效帮助用户解决实际困难，从而将用户转化为忠实粉丝，是我们的重要挑战。
