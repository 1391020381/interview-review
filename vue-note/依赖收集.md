# Watcher
1. new Watcher() 时会触发 get() 当前表达式的执行
2. get()  会先把当前 Watcher push进行 Dep中 然后触发get()
3. pushTarget(this) this.getter() popTarget()

4. update(){ this.get() }
# Dep
1. pushTarget(watcher)  stack.push(watcher) Dep.target = watcher  后面执行 getter() 
* 当Dep.target有值的时候, dep.depend() Dep.target.addDep(this)    也就是在 watcher保存 dep 在调用dep.addSub(this)

* new Watcher()-> get() -> 
* -> pushTarget(Dep.target = watcher)/get(触发 get/set) dep保存 watcher
* -> defineProperty(get)  Dep.target有值   dep.depend()-> Dep.target.addDep(this)
* -> Dep.target.addDep(this)  在 watcher.addDep(dep) -> watcher保存 dep dep.id ->  dep.addSub(this)
* ->  dep.addSub(this)  保存 watcher
* -> defineProperty(set)  触发 sub.update-> watcher.update() ->  this.get() 也就是表达式
